# 云音 · 播放器 & 一图流字幕 PV 工具

一个零依赖、可自部署的网页工具。仿网易云音乐播放器界面，上传 **音频 + 封面 + LRC 歌词** 即可自动播放、时间轴对齐；一键切换到 **一图流字幕 PV 制作台**，用歌词字幕做出 AE 风格的动态排版、关键帧动画、大面积矢量动效与画面滤镜，直接录制导出成片。面向 **翻唱 PV / 歌词写真** 制作。

> - 播放器、PV 制作、导出全部在浏览器里跑，**纯前端即可用**。
> - 可选的 `server.js` 只在你想 **粘贴网易云链接自动解析歌词** 时才需要。
> - 部署与使用详见 [部署教程.md](./部署教程.md) 和 [使用说明.md](./使用说明.md)。

## ✨ 功能一览

### 🎵 播放器模式（仿网易云）
- 上传 **音频 + 封面图 + LRC 歌词** 即可播放
- 封面做成 **黑胶唱片**，随音乐旋转、暂停停转，配唱臂动画
- **歌词时间轴自动对齐**，当前句高亮居中滚动，点歌词跳转
- 自定义 **歌曲名 / 歌手 / 原唱 / 专辑**
- 进度拖动、音量、快退快进、空格/方向键快捷键
- **可录制导出播放器画面为视频**（.webm）

### 🎬 一图流字幕 PV 制作台
- **字幕排版**：居中 / 左右交替 / 随机跳位 / 逐词错落，横排竖排切换，全部自动均衡换行
- **版式模板**（网易云动效歌词风）：字底板 / 关键字强调 / 错落阶梯 / 框字 / 杂志封面，实心色块作字底板、空心图形与线条做破型装饰
- **字幕入场动效**：淡入 / 上浮 / 弹缩 / 聚焦 / 逐字打字 / 逐字弹出 / 侧滑
- **关键帧动画**：字幕整体、背景运镜、每个图形元素都能对 位置/缩放/旋转/透明度 打关键帧，带缓动曲线；独立时间轴精细定位（逐帧、双击输入时间）
- **逐字拆分**：把一句歌词拆成一个个可独立拖动/旋转/缩放/打关键帧的字元素
- **图形元素库**：23 种矢量图形自由布置，独立动效与关键帧
- **矢量图形装饰**：下划线、角括号、卡点圆点、粒子、网格、边框、十字准星、四角框、扫描线、竖边条等
- **大面积矢量动效**（铺满画面）：扩散圆环 / 旋转放射 / 滚动百叶 / 斜纹 / 网格波动 / 漂浮多边形 / 半调点阵 / 律动条 / 边框脉冲 / 光束扫过 / 随机方块 / 故障条 / 多边形爆发 / 随机散布，可调颜色、粗细、大小、透明度、层级
- **画面滤镜 / 整体特效**：暗角、柔光、扫描线、噪点、色调叠色、双色调、负片、色差、老电影、影院黑边、柔焦、暖/冷色调、复古褐、漏光
- **跟拍点击测 BPM**：跟着音乐点几下自动算出 BPM，卡点动效随之律动
- **导出成片**：实时录制画面 + 音频为 .webm

### 🔗 网易云歌词解析（可选，需 server.js）
- 面向翻唱场景：粘贴原曲分享链接 / 歌曲 ID，自动填入 **歌词** 与 **原唱**（原曲演唱者）
- 采用网易云网页端加密接口（WEAPI），比明文接口更稳定，明文接口自动兜底
- 封面与音频用你自己的翻唱素材手动上传，解析不涉及这两项

## 🚀 快速开始

### 只用播放器 / PV（无需 Node，最简单）
把项目下载下来，**双击 `index.html`** 即可使用；或部署到任意静态托管（GitHub Pages / Vercel / Netlify）。素材自行准备：音频、封面、LRC 歌词。

### 需要网易云链接自动解析歌词
需要 Node.js 18+：
```bash
npm install
npm start
```
浏览器打开 `http://localhost:8080`，在编辑面板粘贴网易云链接即可。

> 完整的本地 / 云端部署步骤见 **[部署教程.md](./部署教程.md)**。

## 📝 LRC 歌词格式

标准 LRC：
```
[ti:歌曲名]
[ar:歌手]
[offset:500]
[00:12.34]第一句歌词
[00:15.00][01:20.00]重复出现的副歌   ← 一行多时间戳
```
`[ti:] [ar:] [al:] [by:] [offset:] [length:]` 等标签会被自动识别，`offset` 用于整体微调时间轴。

## ⚠️ 边界说明（诚实）

- **不解析 / 不下载音频**：网易云音频受版权保护，本工具不获取音频，请自行准备并手动上传。解析只提取歌词与原唱这类公开信息。
- **导出格式为 .webm**：浏览器原生录制。需要 mp4 可用 `ffmpeg -i pv.webm pv.mp4` 自行转换。
- **少数读像素的滤镜**（双色调 / 负片 / 色差）在使用跨域图片时可能失效；用本地上传的图片不受影响。
- 仅供学习交流，请尊重音乐版权。
