# 使用说明

打开工具后，顶部有两个模式：**播放器** 和 **一图流字幕 PV**。右上角「＋ 导入 / 编辑」用来上传素材。

---

## 一、导入素材

点右上角「＋ 导入 / 编辑」，弹出面板：

- **音频文件**（必需）：mp3 / wav / m4a 等。
- **封面 / 背景图**：播放器里作黑胶封面，PV 里作背景。
- **LRC 歌词**：可上传 `.lrc` 文件，或直接把歌词文本粘进文本框。
- **歌曲信息**：歌曲名 / 歌手 / 原唱 / 专辑，按需填。
- **网易云链接解析**（需 server.js 启动）：粘贴原曲分享链接或歌曲 ID，点「解析」，自动填入歌词和原唱。封面/音频仍需你自己上传。

填好点「应用」。素材在两个模式间共享，导入一次即可。

---

## 二、播放器模式

- 黑胶唱片随音乐旋转，歌词在右侧滚动、当前句高亮，点歌词可跳转。
- 底部控制条：播放/暂停、快退快进、进度条、音量。
- 快捷键：`空格` 播放/暂停，`←` `→` 快退快进。
- **导出播放器视频**：右下角「录制并导出播放器视频」，会实时录制画面+音频为 `.webm`（时长=歌曲长度，播完自动停，也可手动停）。

---

## 三、一图流字幕 PV 制作台

右侧是控制面板，从上到下：

### 画幅
16:9 横屏 / 9:16 竖屏 / 1:1 方形。

### 字幕字体
字号、字重、垂直位置、主色、描边色与粗细。

### 字幕排版
- **方向**：横排 / 竖排（长句都会自动均衡换行）。
- **排版**：居中 / 版式模板 / 左右交替 / 随机跳位 / 逐词错落。
- **字幕旋转**：整体旋转，可超过 360°（多圈）。
- **✂ 把当前句拆成逐字元素**：见下方「逐字拆分」。

### 版式模板（选「版式模板」后出现）
网易云动效歌词风的设计排版，5 种：字底板 / 关键字强调 / 错落阶梯 / 框字 / 杂志封面。
- 实心色块作单字底板（自动比字大），空心图形与线条做破型装饰，随机挑字加底板。
- 可调整体缩放、错落强度、放大倍数；配色沿用字幕主色/描边/底板色。

### 字幕底板
给整句字幕加：色条 / 方框 / 色块 / 上下线，颜色可选。

### 字幕入场动效
淡入 / 上浮 / 弹缩 / 聚焦 / 逐字打字 / 逐字弹出 / 侧滑，入场时长可调。

### 字幕整体关键帧（位移/缩放/旋转）
让整句字幕做移动/缩放/旋转动画：拖时间轴到某时刻 → 调滑块 → 点 ◆ 打点，至少两帧成动画。缓动方式在「关键帧动画」里选。

---

## 四、关键帧动画（重点）

**背景运镜、字幕、每个图形元素、拆出的每个字**都能打关键帧，属性有 位置X/Y、缩放、旋转、透明度。

操作流程：
1. 用底部**独立时间轴**把播放头拖到某个精确时刻（可点 `‹ ›` 逐帧、`«.1 .1»` 微调，或双击时间数字直接输入秒数）。
2. 调整对象（拖动、改滑块），画面**实时预览**。
3. 点对应的 **◆ 打点** 按钮记录关键帧。
4. 换个时刻，改成不同的值，再打一帧。
5. 至少两帧之间就会自动补间形成动画，带缓动（匀速/慢入/慢出/缓入缓出）。

关键帧清单里点小胶囊可跳到该时刻，点 × 删除。

---

## 五、逐字拆分

在「字幕排版」区点 **✂ 把当前句拆成逐字元素**：当前这句歌词的每个字会变成独立元素，进入图形元素列表。之后每个字都能像图形一样单独 **拖动 / 旋转 / 缩放 / 打关键帧 / 调颜色**，做出精细的逐字动画。原句字幕会自动隐藏，避免重叠。

---

## 六、图形元素（自由布置）

点图形网格里的任意图形（23 种：方/圆/环/三角/星/菱形/箭头/心形/波浪/括号…）添加到画面：
- 直接在左侧画面里**拖动定位**。
- 选中后在下方调：填充/描边色、大小、不透明度、旋转、层级（背景/字幕之下/最上层）。
- 独立动效：卡点脉冲、旋转、抖动、漂浮、呼吸、闪烁。
- 也能对它打位置/缩放/旋转/透明度关键帧。

---

## 七、装饰与特效

- **矢量图形装饰**：下划线、角括号、卡点圆点、粒子、网格、边框、进度条、十字准星、四角框、扫描线、竖边条。
- **大面积矢量动效**（铺满画面，不跟卡点）：扩散圆环、旋转放射、滚动百叶、斜纹、网格波动、漂浮多边形、半调点阵、律动条、边框脉冲、光束扫过，以及带随机性的 随机方块 / 故障条 / 多边形爆发 / 随机散布。可调颜色、粗细、大小、不透明度、层级（字幕前/后）。
- **画面滤镜 / 整体特效**：暗角、柔光、扫描线、噪点、色调叠色、双色调、负片、色差、老电影、影院黑边、柔焦、暖/冷色调、复古褐、漏光。配滤镜颜色与强度。

---

## 八、卡点（BPM）

想让卡点类动效踩上节拍：
1. 播放音乐。
2. 跟着拍子连续点 **「👆 跟拍点击测 BPM」**（至少 4 下）。
3. 工具实时算出 BPM 并应用，起点定在你点的第一下。停 2 秒会自动重新开始。

之后所有卡点脉冲、律动条、圆环等会踩着这个节拍律动。

---

## 九、导出成片

PV 面板底部「● 录制并导出 PV（.webm）」：
- 从头播放并实时录制画面 + 音频，时长 = 歌曲长度。
- 播完自动停止，也可中途手动停。
- 导出 `.webm`；需要 mp4 用 `ffmpeg -i pv.webm pv.mp4` 转换。

> 建议导出前把所有效果、关键帧、卡点都调好，完整播一遍确认无误再录。

---

## 小技巧
- 所有数值滑块都能**双击手动输入**精确值，旁边 `↺` 一键复位。
- 素材两个模式共享，改一次两边都变。
- 效果可叠加，但别一次开太多大面积动效 + 滤镜，画面会太满、也更吃性能。
